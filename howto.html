Background Vocabulary

  1. HTML
  2. tags
  3. <p>
  4. <table>
  5. <td>
  6. <tr>
  7. <div>
  8. <br>
  9. <id class = "test">
  10. <body>
  11. <head>
  12. <title>
  1. HTML stands for Hyper Text Markup Language. It's a language used to create webpages and web apps. HTML was used within the About Me Page, Hardware Page, etc. In HTML, consists of several different tags that are meant for different tasks.
  2. Tags are different elements used for placements and structuring of what goes into the final page. There's a large amount of different tags that can be used to help finalize your page. Below this, are tags I have used and what they are for
  3. The 'p' tag is intially meant for paragraph-hence, the letter chosen.
  4. The table tag is the element used to make tables. In the table tag, you can add multiple table rows, columns, etc.
  5. The 'td' in this tag, stands for 'table data.' Table data is what you put into the table-whether it's numbers, information, etc. It is different from the table tag because the table tag is just meant to tell the computer that you want the inputted information to be in a table format.
  6. The 'tr' in this tag stands for 'table rows.' Table rows are the different rows of a table, obviously. In rows, you can add more tags nested inside for headers or data.
  7. The 'div' tag is a large chunk of code that is given an ID and is used for Javascript and CSS.
  8. The 'br' tag stands for a line break that people want in their text for their web formatting.
  9. An id class is used as a selector for HTML elements. You have to name each id to whatever you want for the computer to know that it is encapsulated into one attribute. Once named, you can go to your CSS file and format the specific id to your liking.
  10. The body tag is essentially everything in your code, put together. Everything that is in your HTML that is shown, is what should be in between your beginning and ending body tag.
  11. The head tag is the tag that you use in HTML for headers. However, although a header is essentially a title, do not get the title tag mixed up with your header tag.
  12. Unlike the header tag, the title tag is used in HTML for naming your tab at the top of your screen. A header tag is used to be shown in your website-your title tag is used at the top to name the site.
